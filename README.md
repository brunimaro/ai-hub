# 🤖 AI Hub — Portail d'Intelligence Artificielle

## 📋 Description

**AI Hub** est une landing page moderne et élégante qui centralise l'accès aux principaux modèles d'intelligence artificielle. Idéal pour naviguer rapidement entre différents outils selon vos besoins : conversation, recherche, code, création d'images, etc.

✨ **Design** : Interface sombre, animations fluides, effets de glow interactifs  
⚡ **Performance** : 100% statique, chargement instantané, zéro dépendance  
📱 **Responsive** : Optimisé mobile, tablette et desktop

## 🌟 Fonctionnalités

| Fonctionnalité | Description |
|----------------|-------------|
| 🔍 **Recherche instantanée** | Filtrez les IA par nom ou description en temps réel |
| 🏷️ **Filtres par catégorie** | Conversation • Recherche • Code • Image • Open Source |
| 🎨 **Effets visuels** | Glow interactif au survol, orbes animés en arrière-plan |
| ⌨️ **Raccourci clavier** | Appuyez sur `/` pour focaliser la recherche |
| 📱 **Mobile-first** | Navigation tactile optimisée |
| 🔗 **Liens directs** | Ouverture dans un nouvel onglet avec `target="_blank"` |
